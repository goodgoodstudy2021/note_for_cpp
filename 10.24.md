# How to create/instantiate object in c++
    堆的东西会放到我去删除掉
    栈的东西有生命周期
使用new字符定义的变量和函数会存储在堆上，但必须记得使用完毕之后，使用的detele将其删除掉，不然它还会继续存在内存中。
栈的空间比较小，如果需要的空间比较大的话，可以使用new字符来使用堆上的内存
# new keyword
